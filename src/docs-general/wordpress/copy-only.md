# Copy Only

> A module wich allows users to have some copy

## Configuration 
- Alignment
	- There are 3 options, these being Center, Left & Right
- Theme
	- Returns one of 3 options, this ensures that all of the sections remain on brand.
	- primary
 	- secondary
	- alternative
- Background Images
	- Landscape
	- Portrait

## Options
- Header
- Sub Header
- Copy
- CTAs (up to 3)
